## 🚀 内容
引入 react-redux
技术上讲你可以直接使用 store.subscribe() 来编写容器组件。
但不建议这么做的原因是无法使用 React Redux 带来的性能优化。
也因此，不要手写容器组件，而使用 React Redux 的 connect() 方法来生成，后面会详细介绍。
